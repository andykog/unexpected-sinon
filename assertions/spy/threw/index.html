    Passes if the spy threw the given exception.

    The exception can be a string denoting its type, or an actual object. If only one argument is provided, the assertion passes if the spy ever threw any exception.

    var stub = sinon.stub();
    var error = new TypeError('wat');
    stub.throws(error);
    try { stub(); } catch (e) {}
     
    expect(stub, 'threw');
    expect(stub, 'threw', 'TypeError');
    expect(stub, 'threw', error);

    In case of a failing expectation you get the following output:

    expect(sinon.spy(), 'threw', 'SyntaxError');
    spy did not throw exception
